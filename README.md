# 简介

编写一个最好的人民币大写转换库。
以 C# 为基础，并改编成其他语言版本。

目前支持的语言有：

* C#
* C++
* C
* HaXe
* Java
* Javascript
* Lua
* Python
* Ruby
* Rust
* Go  //由网友 @henry.chen & @deepzz0 贡献，因为我很讨厌 go，怎么可能写 go 的程序啦

计划支持：

* Swift
* D
* Haskell
* Clojure
* Scheme
* Erlang
