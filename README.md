#简介

编写一个最好的人民币大写转换库。
以 C# 为基础，并改编成其他语言版本。

目前支持的语言有：

* C#
* C++
* C
* HaXe
* Java
* Javascript
* Lua
* Python
* Ruby

计划支持：

* Rust
* D
* Haskell
* Clojure
* Scheme
* Go
* Erlang
